## 模式分类：
### 创建型
> 创建型模式涉及到将对象实例化，这类模式都提供一个方法，将客户从所需要的实例化的对象中解耦。
- 单件模式
- 工厂模式
- 抽象工厂模式
- 建造者模式
- 原型模式

### 行为型
> 只要是行为型模式，都涉及到类和对象如何交互及分配职责。
- 模板方法模式
- 命令模式
- 迭代器模式
- 观察者模式
- 状态模式
- 策略模式
- 责任链模式
- 解释器模式
- 中介者模式
- 备忘录模式
- 访问者模式

### 结构型
>结构型模式可以让你把类或对象组合到更强大的结构中
- 装饰者模式
- 适配器模式
- 代理模式
- 外观模式
- 组合模式
- 桥接模式
- 蝇量模式
